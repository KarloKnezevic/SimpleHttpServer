Educational Runtime Playground

SimpleHttpServer

A from-scratch Java HTTP server that demonstrates parsing, routing, static serving, templating, and a custom script runtime (.smscr).

HTTP 1.1 Parsing Route Matching Session Cookies Script Engine

Core HTTP Routes

Hello Route

Minimal route handler returning plain text.

GET /hello

Echo Route

Shows query parsing and JSON response building.

GET /echo?x=1&x=2&name=Alice

Template Route

Renders HTML and tracks session visits.

GET /template?name=Alice

Path Params

Demonstrates parameterized URL matching.

GET /users/42

Real-World Script Demos (.smscr)

Invoice Generator

Calculates subtotal, discount, tax, and grand total from query params.

GET /scripts/invoice.smscr

Campaign KPI Report

Computes CTR, conversion rate, CPC, and CPA from campaign metrics.

GET /scripts/campaign-report.smscr

Timesheet Summary

Estimates project billing and prints a weekly breakdown using a FOR loop.

GET /scripts/timesheet.smscr

Support Ticket View

Returns HTML and tracks per-session ticket open count via persistent params.

GET /scripts/support-ticket.smscr

Mock API Health

Returns JSON and increments a session-scoped request counter.

GET /scripts/api-health.smscr
